# ELXR AI Mentor Iframe Embed Snippets

Copy and paste these iframe snippets into your website to embed the AI mentors.

**Replace `YOUR-REPLIT-DOMAIN.replit.app` with your actual published domain.**

## 1. Mark Kohl
**Expertise:** Psychedelics, Spirituality, Fungi, Kundalini

### Standard Size (800x600)
```html
<iframe 
  src="https://YOUR-REPLIT-DOMAIN.replit.app/avatar?mentor=mark-kohl" 
  width="800" 
  height="600" 
  frameborder="0" 
  allow="camera; microphone; autoplay; encrypted-media; fullscreen" 
  allowfullscreen
  title="Chat with Mark Kohl">
</iframe>
```

## Usage Notes

### Required Permissions
All iframes include the `allow` attribute with these permissions:
- `camera` - For video features (optional)
- `microphone` - For voice input (optional)
- `autoplay` - For avatar video playback
- `encrypted-media` - For secure media streaming
- `fullscreen` - To enable fullscreen mode

### Customization Options

#### Custom Dimensions
You can adjust `width` and `height` to fit your layout:
```html
<iframe 
  src="https://YOUR-REPLIT-DOMAIN.replit.app/avatar?mentor=mark-kohl" 
  width="1024" 
  height="768" 
  ...
</iframe>
```

#### Responsive Container
For better responsive behavior, wrap the iframe in a container:
```html
<div style="position: relative; width: 100%; padding-bottom: 75%; /* 4:3 aspect ratio */">
  <iframe 
    src="https://YOUR-REPLIT-DOMAIN.replit.app/avatar?mentor=mark-kohl" 
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: none;" 
    allow="camera; microphone; autoplay; encrypted-media; fullscreen" 
    allowfullscreen
    title="Chat with Mark Kohl">
  </iframe>
</div>
```

### Getting Your Domain
1. Publish your app using the "Deploy" button in Replit
2. Your domain will be: `your-project-name.replit.app`
3. For custom domains, configure them in your Replit deployment settings
4. Replace `YOUR-REPLIT-DOMAIN.replit.app` in all snippets with your actual domain

### Testing
Before deploying to production:
1. Test each iframe in a local HTML file
2. Verify all mentor IDs work correctly
3. Check responsive behavior on mobile devices
4. Ensure fullscreen functionality works
5. Confirm audio/video permissions are requested properly

## Support
If you encounter issues with the embeds, verify:
- ✅ Domain is correct and app is published

- ✅ Mentor ID matches exactly (case-sensitive)
