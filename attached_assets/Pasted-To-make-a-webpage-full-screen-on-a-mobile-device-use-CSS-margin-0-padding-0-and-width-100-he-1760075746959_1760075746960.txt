To make a webpage full screen on a mobile device, use CSS margin: 0; padding: 0; and width: 100%; height: 100%; on html, body and any containers, and then trigger the Fullscreen API with JavaScript's element.requestFullscreen() in response to a user interaction, or use the Add to Home Screen feature in some browsers for a standalone app-like experience. 
1. Fullscreen CSS (for a full-width, full-height page)
This method doesn't use the browser's actual fullscreen mode but makes the content take up the entire viewport.
HTML: Ensure your HTML is structured to have a main content container.
Code

    <html>
    <head>
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Fullscreen Page</title>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <div class="content-wrapper">
            <!-- Your page content here -->
        </div>
    </body>
    </html>
CSS (style.css): Apply these styles to remove default margins and paddings and set the width and height to 100%. 
Code

    html, body {
        margin: 0;
        padding: 0;
        width: 100%;
        height: 100%;
        overflow: hidden; /* Prevents scrolling */
    }

    .content-wrapper {
        width: 100%;
        height: 100%;
        /* Optional: Use flexbox to center content */
        display: flex;
        justify-content: center;
        align-items: center;
    }
2. Fullscreen API (for true, browser-level fullscreen)
This allows you to make a specific element or the whole page go into the browser's fullscreen mode, hiding toolbars and status bars. 
HTML: Add a button to trigger the fullscreen action.
Code

    <button id="fullscreen-btn">Go Fullscreen</button>
    <div id="fullscreen-element">
        <!-- Content you want to display in fullscreen -->
    </div>
JavaScript: Use the requestFullscreen() method to enter fullscreen mode. This must be triggered by a user event, like a button click. 
JavaScript

    const fullscreenElement = document.getElementById('fullscreen-element');
    const fullscreenButton = document.getElementById('fullscreen-btn');

    fullscreenButton.addEventListener('click', () => {
        if (fullscreenElement.requestFullscreen) {
            fullscreenElement.requestFullscreen();
        }
        // Or for the entire document:
        // document.documentElement.requestFullscreen();
    });

    // To exit fullscreen (e.g., via the Esc key or a button):
    // document.exitFullscreen();
